# CachyOS Installation Guide

A step-by-step guide for installing CachyOS (Arch-based, performance-optimized) and setting it up for gaming on **AMD hardware**.

## What's covered

| # | Topic |
|---|---|
| 0 | Prerequisites & backups |
| 1 | Downloading the ISO + creating a bootable USB (Rufus) |
| 2 | BIOS settings (Secure Boot, UEFI boot) |
| 3 | Installation via Calamares (erase disk / dual-boot / manual) |
| 4 | First steps after boot (system update, CachyOS Hello) |
| 5 | GPU / drivers — AMD (Mesa/RADV, no manual install needed) |
| 6 | Gaming — Steam + GE-Proton via ProtonUp-Qt |
| 7 | AMD graphics control (CoreCtrl, MangoHud, GameMode, OBS) |
| 8 | RGB keyboard (OpenRGB) |
| 9 | Bonus tweaks (GameMode, fstrim, kernel params) |
| 10 | Dev tools (git, SSH key for GitHub, paru AUR helper, VS Code) |
| 11 | Fix: Code - OSS / VS Code freezes on sign-in (Electron bug workaround) |

## Reference hardware

- CPU: AMD Ryzen 5 7600X
- GPU: AMD Radeon RX 6800 (RDNA2)
- RAM: 32 GB
- SSD: Kingston KC3000 NVMe
